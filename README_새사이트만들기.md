# 🌐 새 깃허브 사이트 업로드 묶음

이 폴더(`깃허브_새사이트`)를 **통째로** 새 저장소에 올리면 바로 작동합니다. 이미지가 상대경로(`images/...`)라 저장소 이름·주소와 상관없이 동작해요.

## 만드는 방법
1. 깃허브에서 **New repository** 로 새 저장소를 만듭니다.
   - 사이트 주소를 `아이디.github.io` 로 하고 싶으면 저장소 이름을 **`아이디.github.io`** 로 만드세요.
   - 그게 아니면 아무 이름이나 OK (주소는 `아이디.github.io/저장소이름/` 형태가 됩니다).
2. **Add file → Upload files** 클릭.
3. 이 폴더 안의 **index.html** 과 **images 폴더**를 함께 끌어다 놓습니다.
4. **Commit changes** → 저장소 **Settings → Pages** 에서 Branch를 `main`으로 지정 → 저장.
5. 잠시 후 `https://아이디.github.io/...` 로 접속하면 웹교안이 보입니다.

## 폴더 구성
```
깃허브_새사이트/
├─ index.html      ← 웹교안 본체 (이미지 상대경로)
└─ images/         ← 사용 이미지 15개
```
