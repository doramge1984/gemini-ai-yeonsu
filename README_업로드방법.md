# 🌐 웹교안 깃허브 업로드 묶음 (최신본)

이 폴더(`깃허브_새사이트`)를 통째로 올리면 바로 작동합니다.

## 들어 있는 파일
- index.html  (웹교안 본체)
- images/  (사용 이미지 18개)
- 여행엽서_자동화_교안.html  (STEP 9에서 불러오는 교안)

## 업로드 방법
1. github.com 로그인 → 우측 상단 **+ → New repository**.
2. 사이트 주소를 `아이디.github.io`로 하려면 저장소 이름을 **아이디.github.io**로 (예: doramge1984.github.io). 그 외 이름도 가능.
3. **Public** 선택 → **Create repository**.
4. **uploading an existing file** 클릭 → 이 폴더 안의 **index.html · images 폴더 · 여행엽서_자동화_교안.html** 를 모두 끌어다 놓기.
5. **Commit changes**.
6. 저장소 **Settings → Pages** → Branch를 **main / (root)** 로 지정 → **Save**.
7. 1~2분 뒤 `https://아이디.github.io/저장소이름/` 으로 접속하면 웹교안이 보입니다.
